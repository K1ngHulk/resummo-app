# Project Instructions

## Tool usage policy

Do not use browser, Playwright, visual QA, screenshots, browser automation, or live UI inspection by default.

Use those tools only when the user explicitly asks for visual inspection, browser review, Playwright validation, Figma review, screenshot comparison, or responsive/browser QA.

For normal implementation tasks, prefer code inspection, targeted edits, `npm.cmd run lint`, and `npm.cmd run build`.

Do not spend tokens on browser-based validation unless it is specifically requested or clearly required to reproduce a visual/browser-only bug.
